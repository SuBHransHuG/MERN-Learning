The span tag is an inline element by default in HTML, which means that it only takes up as much width and height as its content requires. Inline elements do not respect the width and height CSS properties because they flow within the normal line of text.
To apply width and height to a span, you need to change its display property to something that can accommodate these dimensions, such as inline-block or block.
